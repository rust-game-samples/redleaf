WordPress parity タスクを `ai_docs/wordpress-parity-tasks.md` に追加します。

追加するタスク: $ARGUMENTS

## 手順

1. `ai_docs/wordpress-parity-tasks.md` を読んで現在の構造を把握する
2. `$ARGUMENTS` が空の場合:
   - 未完了タスク（`- [ ]`）をフェーズ別にリストアップして表示する
   - 「追加したいタスクを `/wp-add-task <説明>` で指定してください」と案内する
3. `$ARGUMENTS` が指定されている場合:
   - タスクの内容から最も適切なフェーズ・セクションを判断する
   - 既存の類似タスクがないか確認する（重複防止）
   - 該当セクションに `- [ ] {タスクの説明}` として追記する
   - 新しいセクションが必要な場合は適切な位置に作成する
   - 追加した箇所をユーザーに報告する（フェーズ名 + タスク内容）

## 記載ルール

- タスクは具体的・実装可能な粒度で書く（「～追加」「～実装」「～対応」など動詞で終わる）
- 関連するサブタスクはインデントせず同じ階層でグループ化する
- 実装の参考となる技術名・クレート名・パス名があれば括弧内に記載する
